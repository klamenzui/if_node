# if_node

Installation: 
1. get 'install.sh' script
 - wget https://raw.githubusercontent.com/klamenzui/if_node/main/monitoring/install.sh -O ~/monitoring/install.sh
 - chmod +x ~/monitoring/install.sh
2. Run install.sh
 - cd ~/monitoring/
 - ./install.sh
3. Import 'grafana.json' into your grafana

done! :)
